# Financial NER with spaCy

This project trains a custom Named Entity Recognition (NER) model using spaCy 
to extract financial entities such as ORG, PERSON, MONEY, DATE, PERCENT, and GPE.

## Files
- task1.ipynb → Colab notebook with training & testing code
- training_data.json → Annotated dataset
- input.txt → Sample test input

## How to run
Open task1.ipynb in Google Colab and execute all cells.
